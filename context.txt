# Chiya Sangeet Restaurant Website

A modern website for Chiya Sangeet restaurant featuring an interactive menu and location information.

## Overview

This website provides customers with an easy way to browse our menu items and find our location. It also includes an admin interface for staff to manage menu content.

## Features

- Interactive menu display system
  - Supports up to 50 menu items
  - Each item includes photo, name and price
  - Modern, consistent styling throughout

- Dual mode interface
  - Public viewer mode for customers
  - Secure admin mode for staff
  
- Admin capabilities
  - Edit menu item names
  - Update prices
  - Change item photos
  - All changes preserve existing styling

- Location
  - Google Maps integration
  - Easy to find our restaurant

## Technology

Built with modern web technologies and an emphasis on clean, professional design that maintains visual consistency across all pages and components.
